# SongLens 1.0.8

## Filter And Song Grid

- Fixed the View Filter dialog so `View All Songs` remains selected instead of reverting to the remembered 30-day filter.
- Fixed startup restoration so `View All Songs` takes precedence over a dormant age-filter value.
- Added `Display filter results in the Song Grid`, allowing age and date filters to populate the complete matching-song list.
- Saved the new Song Grid display preference and restored it with the filter session.
- Kept the active age or date filter visible in the status bar when its complete results are displayed in the Song Grid.
- Tightened the View Filter layout and grouped the Song Grid option beneath a separator.

## UI

- Matched the `Change Columns` mouseover color to the `Snapshot` button.
