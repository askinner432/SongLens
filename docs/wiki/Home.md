# SongLens

SongLens is a Windows desktop app for browsing Studio One `.song` files, inspecting metadata, reviewing tracks and mixer details, and exporting useful song information without opening every project in Studio One.

It is designed for composers, producers, and power users who want a fast way to search a song library and inspect what is inside a project at a glance.

## What SongLens Does

- Browses a Studio One songs folder with a filtered folder tree
- Reads `.song` files locally without requiring network access
- Shows summary metadata such as title, artist, tempo, key, time signature, Studio Version, and file dates
- Displays filtered raw attributes from `metainfo.xml`
- Lists tracks, instruments, track notes, and groups
- Shows mixer data in separate Main, Inserts, and Sends sections
- Reads song notes from `notes.txt`
- Opens the song History browser for saved history copies
- Searches across song metadata, notes, and track details
- Exports search results or library views to CSV
- Creates musician-friendly song snapshots in Text or JSON

## Main Window Overview

SongLens is organized into three main areas:

- Folder tree on the left
- Song grid on the upper right
- Detail tabs on the lower right

The app also includes:

- A top path and search area
- Toolbar buttons for folder and browsing actions
- A `Recently Viewed` drop-down above the song grid
- Menus for export, filtering, preferences, and advanced features

## Common Tasks

- Browse to a songs folder and inspect projects without opening Studio One
- Search for songs by title, artist, tempo, key, notes, track names, and more
- Review track instruments, groups, mixer inserts, mixer sends, and mixer main chains
- Export the current view to CSV
- Save a snapshot of a song in Text or JSON format
